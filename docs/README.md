## What is This?

This is a full-stack real-time chat application where users can:
- Register and login
- Search for other users
- Send messages in real-time
- Create and manage group chats
- Get notifications for new messages

## Tech Stack

**Frontend:**
- React + Vite
- Redux Toolkit
- Socket.IO Client
- Tailwind CSS

**Backend:**
- Node.js + Express
- MongoDB + Mongoose
- Socket.IO
- JWT Authentication
